2. ステージ生成（ステージデータ + 乱数シードを基に）
3. ゲーム開始
4. ゲーム終了

## クラス
- User : ユーザーのパラメータ
    - Decoration : 装飾
        - Head
        - Body
        - Reg
        - Arm
    - Param : レースに影響するパラメータ
    - other


- Stage : ステージ全体のデータ- 
    - section : 各区間ごとのデータ
    - tileset : IDに一致する画像データ
- UI : UI 全般
